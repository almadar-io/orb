# Internationalization (i18n)

Orb supports writing schemas in your native language. The CLI automatically detects the language based on keywords and translates it to the standard internal representation before compilation.

<OrbitalDiagram />

## Usage

Simply write your schema using the translated keywords and operators for your language. The file extension remains `.orb`.

### Example

```lolo
;; app TaskApp

orbital TaskManager {
  entity Task {
    id : string!
    title : string
    completed : boolean
  }
  trait TaskBrowser -> Task [interaction] {
    state active {
      INIT -> active
        (fetch Task)
        (render-ui main { type: "entity-table", entity: "Task", fields: ["title", "completed"], itemActions: [{ event: "COMPLETE", label: "Complete" }] })
      COMPLETE -> active
        (set @entity.completed true)
    }
  }
  page "/tasks" -> TaskBrowser
}
```
