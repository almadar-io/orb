# Prispevanje k Orb

Dobrodosli v skupnosti Almadar! Veseli smo, da prispevate.

## Nacini prispevanja

### Dokumentacija

- Popravite tipkarske napake in izboljsajte jasnost
- Prevedite vsebino v arabscino ali druge jezike
- Pisite vadnice in vodice
- Dodajte primere in primere uporabe

### Standardna knjiznica

- Predlagajte nove operatorje
- Izboljsajte obstojece implementacije
- Dodajte teste in preizkuse zmogljivosti

### Vzorci in lastnosti

- Oddajte nove vzorce UI
- Ustvarite ponovno uporabne lastnosti
- Dokumentirajte najboljse prakse

### Skupnost

- Odgovarjajte na vprasanja na Discord
- Pisite objave na blogu
- Ustvarjajte video vadnice
- Imejte predavanja na srecah

## Postopek prispevanja

### 1. Za dokumentacijo

```bash
# Razvejite in klonirajte repozitorij
git clone https://github.com/YOUR_USERNAME/almadar.git

# Ustvarite vejo
git checkout -b docs/improve-getting-started

# Naredite spremembe
# ...

# Oddajte zahtevo za zdruzevanje
```

### 2. Za kodo (standardna knjiznica, vzorci)

1. **Odprite RFC** - za pomembne spremembe zacnite razpravo
2. **Pridobite mnenja** - pocakajte na prispevke skupnosti
3. **Implementirajte** - ko je odobreno, oddajte PR
4. **Pregled** - naslovite povratne informacije vzdrzevalcev
5. **Zdruzitev** - praznujte!

### 3. Za prevode

Uporabljamo delovni tok za prevode:

```bash
# Arabski prevodi gredo v mapo /ar/
docs/
├── en/
│   └── getting-started/
│       └── introduction.md      # Vir
└── ar/
    └── getting-started/
        └── introduction.md      # Prevod
```

**Smernice za prevajanje:**
- Ohranjajte doslednost tehnicnih izrazov (glej slovarcek)
- Ohranite blokov kode v izvornem jeziku
- Uporabljajte oblikovanje, ki uposteva RTL
- Testirajte upodabljanje pred oddajo

## Kodeks ravnanja

### Nasa zaobljuba

Zavezani smo zagotavljanju prijetnega in vkljucujocega okolja za vse, ne glede na:

- Ozadje ali raven izkusenj
- Spolno identiteto ali izrazanje
- Geografsko lokacijo
- Jezik
- Vero

### Pricakovano obnasanje

- Bodite spostljivi in vkljucujoci
- Dobrodoslite nove clane
- Dajajte konstruktivne povratne informacije
- Osredotocite se na zadevo, ne na osebo

### Nesprejemljivo obnasanje

- Nadlegovanje ali diskriminacija
- Trollanje ali zaljiive opombe
- Objavljanje zasebnih informacij drugih
- Drugo ravnanje, ki se steje za neprimerno

### Uveljavljanje

Krsitve lahko privedejo do:
1. Opozorila
2. Zacasne prepovedi
3. Trajne prepovedi

Prijavite tezave na: conduct@almadar.io
